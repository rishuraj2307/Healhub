# HealHub – Hospital Management System (Spring Boot Backend)

## 📌 Overview
HealHub is a backend-driven Hospital Management System developed using Java and Spring Boot.  
It provides RESTful APIs for managing doctors, patients, appointments, and medical records with proper relational database integration.

---

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Data JPA (Hibernate)
- PostgreSQL
- Maven
- Postman (API Testing)

---

## 🚀 Features
- Doctor Management APIs
- Patient Management APIs
- Appointment Scheduling
- Medical Records & Prescription Handling
- Layered Architecture (Controller → DAO → Entity → Exception)
- Custom Exception Handling
- Relational Database Design

---

## 🗄 Database Structure
Tables:
- appointment
- department
- doctor
- patient
- prescription
- record

---

## 🧪 API Testing
All APIs were tested using Postman with successful 200 OK responses.

---

## 📂 Project Structure
- controller
- dao
- entity
- exception
- dto

---

## 🎯 Purpose
This project demonstrates backend architecture design, REST API development, database integration, and clean project structuring using Spring Boot.
